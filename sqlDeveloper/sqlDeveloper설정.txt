
  :: SQL Developer 설정

   - 도구 > 환경설정 선택,

   1) 환경 > 인코딩 : UTF-8

   2) 데이터베이스 > 고급 : 배경색을 사용하여 널표시 : YELLOW

   3) 데이터베이스 > NLS > 날짜언어 : KOREAN 을 ENGLISH로
   ※ 교재 예제중 JAN, DEC등 영문 날짜를 SQL 엔진이 처리못해서 에러가 나는 경우

   4) 데이터베이스 > 객체 뷰어 : 한번 눌러 객체열기 (체크해제)

   5) 코드 편집기 > PL/SQL 구문 색상 : 구성표-색상테마중 하나 

  ※ 필요시 sqldeveloper color theme를 추가 설치 (Dracular)
 
   6) 코드 편집기 > 글꼴 : 글꼴크기 16 이상,
                           글꼴이름 코딩폰트

   
   7) 코드 편집기 > 행 여백 : 행번호 표시 체크


   8) 코드 편집기 > 형식 : 들여쓰기 공간 4 -> 2 (좁게)
